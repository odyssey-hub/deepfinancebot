Добро пожаловать!
Список доступных команд:
/ticker - получение последней информации по тикеру
/tickers - список доступных тикеров акций
/corr - корреляция между столбцами одного тикера / корреляция между несколькими тикерами
/mean - среднее арифметическое по столбцу(ам) тикера
/var - дисперсия по столбцу(ам) тикера
/median - медиана по столбцу(ам) тикера
/std - среднеквадратическое отклонение по столбцу(ам) тикера
/index - список входящих в индекс тикеров с весами
/indexes - список индексов, в которые входит тикер
/forecast - прогнозирование котировок по тикеру
/delta2w - запуск модели для перебора комбинации указанных тикеров
/delta2w_full - запуск модели для перебора комбинации всех имеющихся тикеров
/export - экспорт исторических данных тикера за определенный период в xlsx
/chart - построение свечного/линейного графика по тикеру(ам) за определенный период
С помощью ключа (-h) или ввода команды без параметров можно получить подробную информацию по команде (например, /corr или /corr -h)
