/var тикер -p период [-t таймфрейм] [-c столбец]
Дисперсия по столбцу(ам) тикера
Параметры:
*тикер
*период (-p) – либо за последние n, либо указанный промежуток
таймфрейм (-t) – по умолчанию дневной
столбец (-c) – по умолчанию CLOSE (!только один!)
Примеры:
/var SBER -p 2w -t h – дисперсия по SBER за последние 2 недели по столбцу CLOSE, таймфрейм часовой.
